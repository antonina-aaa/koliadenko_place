# 📊 Vector Operations & JSON Data Processing (DZ_1)
## 🚀 Основні функції
Програма виконує наступні кроки:
1. **Генерація даних:** Створює файл `numbers.txt` з 1000 випадкових чисел.
2. **Групування:** Зчитує числа та вираховує суму груп по `n` елементів (кількість задає користувач).
3. **Алгоритми сортування:** - Сортування бульбашкою за зростанням (`sortAsc`).
   - Сортування бульбашкою за спаданням (`sortDesc`).
4. **Робота з дублікатами:** - Функція `removeDuplicates` створює масив унікальних значень.
   - Функція `printDifference` виводить елементи, що повторюються.
5. **Збереження результатів:**
   - Математичні результати записуються у текстовий файл `e.txt`.
   - Всі розрахунки (суми, різниці, масиви V1 та V2) зберігаються у `data.json`.
## 🛠 Технології та бібліотеки
* **Мова:** C++
* **Бібліотеки:** * `nlohmann/json` — для роботи з форматом JSON.
    * `vector`, `fstream`, `ctime` — стандартні бібліотеки C++.
* **Заголовні файли:** `vector.h` — містить звичайні функції обробки векторів.

## 📂 Структура проєкту
* `Source.cpp` — головний файл з логікою програми.
* `vector.h` — оголошення та реалізація функцій сортування та обробки.
* `dependencies/` — папка з бібліотекою `nlohmann/json`.
* `data.json` — результат виконання програми у зручному для читання форматі.
