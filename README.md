
# 🛍️ Clothing Store Console App

![C++](https://img.shields.io/badge/C++-17-blue.svg)
![Status](https://img.shields.io/badge/Status-Completed-success)

**Clothing Store Console App** — це консольна програма на мові C++, яка імітує роботу системи оформлення замовлень у магазині одягу. Проєкт демонструє роботу з логікою знижок, обробкою вводу та математичними розрахунками.

---

## 📖 Опис проєкту

Програма дозволяє користувачеві переглядати асортимент магазину, вибирати товари та отримувати автоматичний розрахунок вартості з урахуванням динамічних та статичних знижок.

### 🚀 Основний функціонал

* 🛒 **Каталог товарів:** Перегляд списку актуальних товарів та цін.
* 📅 **Знижка дня:** Система автоматично призначає знижку **-10%** на один випадковий товар залежно від введеної дати.
* 🎟 **Система промокодів:** Можливість додатково знизити ціну за допомогою секретних кодів.
* 💰 **Розумний кошик:** Автоматичний підрахунок фінальної суми в реальному часі.

Ви можете застосувати один із наступних промокодів під час оформлення замовлення:

| Промокод | Знижка |
| :--- | :--- |
| **SALE10** | `-10%` |
| **SALE20** | `-20%` |
| **SUPER30** | `-30%` |

## 🛠 Технології та інструменти

Для розробки було використано сучасний стек для консольних додатків на Windows:

* **Мова програмування:** [C++](https://isocpp.org/) (стандарт **C++17**)
* **Середовище розробки (IDE):** Visual Studio 2022
* **Стандартні бібліотеки:**
    * `<iostream>` — для введення/виведення даних.
    * `<string>` — для роботи з текстовими даними та промокодами.
    * `<iomanip>` — для гарного форматування цін (десяткові дроби).
