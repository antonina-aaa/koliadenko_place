# 🛍️ Clothing Store Console App

🚀 **Основні функції**
Програма імітує простий магазин одягу:

**Відображення товарів:**
Виводиться список товарів з цінами:

* T-shirt, Jeans, Jacket, Sneakers, Hoodie

**Вибір товарів:**
Користувач обирає товар за номером
(ввід `0` — завершення покупки)

**Обробка вибору:**

* перевірка коректності введення
* додавання ціни товару до загальної суми

**Підрахунок суми:**
Обчислюється загальна вартість покупки (**total**)

---

Мова: C++
