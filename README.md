
🛍️ Clothing Store Console App








📖 Опис

Clothing Store Console App — консольна програма на C++, яка імітує роботу простого магазину одягу.

Програма дозволяє:

🛒 обирати товари
📅 отримувати знижку дня (один товар зі знижкою -10%)
🎟 застосовувати промокоди
💰 автоматично підраховувати фінальну суму покупки
🚀 Основний функціонал
🛒 1. Відображення товарів

Список доступних товарів з цінами:

T-shirt
Jeans
Jacket
Sneakers
Hoodie
🎯 2. Вибір товарів
Користувач обирає товар за номером
Ввід 0 — завершення покупки
📅 3. Знижка дня
Введіть дату у форматі dd.mm
Програма обирає один товар зі знижкою -10%
🎟 4. Promo Code System
Промокод	Знижка
SALE10	-10%
SALE20	-20%
SUPER30	-30%

Промокод застосовується до всієї покупки

💰 5. Підрахунок суми
Перевірка коректності введення
Додавання вартості товарів до загальної суми
Вивід фінальної вартості
