
🛍️ Clothing Store Console App
📖 Опис

Clothing Store Console App — це консольна програма на C++, яка імітує роботу простого магазину одягу.

Користувач може обирати товари, отримувати знижку дня та застосовувати промокоди.
Програма автоматично обчислює фінальну вартість покупки.

🚀 Основні функції
🛒 Відображення товарів

Програма показує список товарів із цінами:

T-shirt
Jeans
Jacket
Sneakers
Hoodie
🎯 Вибір товарів
Користувач обирає товар за номером
Ввід 0 — завершення покупки
📅 Знижка дня
Користувач вводить дату у форматі:
dd.mm
Програма визначає один товар зі знижкою -10%
🎟 Промокоди
Промокод	Знижка
SALE10	-10%
SALE20	-20%
SUPER30	-30%

📌 Застосовуються до всієї суми покупки

💰 Підрахунок суми
Перевірка коректності введення
Додавання вартості товарів до загальної суми
Вивід фінального результату
🛠 Технології
Мова: C++
Стандарт: C++17
📚 Використані бібліотеки:
iostream — введення та виведення даних
📂 Структура проєкту
project/
└── clothingstore.cpp
▶️ Як запустити
Скомпілювати програму
Запустити виконуваний файл
Ввести дату (формат dd.mm)
Обирати товари за номером
