# 🛒 Self-Checkout System (Store-Zhovkva)

Ця програма на C++ імітує роботу системи самообслуговування в магазині. Проєкт розроблений як навчальна робота з фокусом на ООП та базові алгоритми.

## ✨ Основні можливості
* **Керування товарами:** додавання продуктів у кошик.
* **Система знижок:** розрахунок фінальної вартості з урахуванням промокодів.
* **Інтерактивність:** вибір магазину (Pet Store або Game Store) через консольне меню.
* **Звукові ефекти:** програма відтворює звук успіху (`success.wav`) при завершенні покупки.

## 🛠 Технології
* **Мова:** C++
* **Середовище:** Visual Studio 2022
* **Бібліотеки:** `<iostream>`, `<string>`, `<vector>`, `<windows.h>` (для відтворення звуку).

## 📂 Структура проєкту
* `Source.cpp` — головна логіка та точка входу.
* `Store.h / .cpp` — клас для керування асортиментом магазину.
* `Product.h / .cpp` — клас опису товару.
* `Discount.h / .cpp` — логіка застосування знижок.
* `Receipt.h / .cpp` — формування та виведення чека.
* `success.wav` — аудіофайл успішної транзакції.
