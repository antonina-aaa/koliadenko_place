# 🧪 Проєкт: Вектори та Unit-тести (C++)

Цей проєкт присвячений тестуванню за допомогою **Microsoft Native Unit Test Framework**.

## 🚀 Основні функції
У файлі `vector.h` реалізовано наступні алгоритми:
- **`sortAsc`** — Сортування за зростанням (Bubble Sort).
- **`sortDesc`** — Сортування за спаданням.
- **`sumOfElements`** — Розрахунок загальної суми елементів.
- **`removeDuplicates`** — Створення вектора з унікальними значеннями.
- **`printDifference`** — Вивід повторюваних елементів.

## 🛠 Технології
- **Мова:** C++
- **Бібліотеки:** `nlohmann/json` (для роботи з даними у форматі JSON).
- **Тестування:** Microsoft CppUnitTest.

## 🧪 Опис Unit-тестів
Проєкт містить 5 автоматичних тестів, які перевіряють:
1. **Сортування:** коректність перестановки чисел.
2. **Математику:** правильність обчислення суми.
3. **Унікальність:** чи правильно видаляються дублікати.
4. **JSON:** чи коректно підключена та працює зовнішня бібліотека.
Усі тести проходять успішно (Status: **Passed** ✅).
## 📂 Структура проєкту
- `dz_1/` — основний код програми та логіка векторів.
- `UnitTest1/` — проект з тестами.
- `data.json` — приклад виводу результатів у форматі JSON.
