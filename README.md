
🛍️ Clothing Store Console App








📖 Опис

Clothing Store Console App — це консольна програма на C++, яка імітує простий магазин одягу.

Користувач може:

обирати товари
отримувати знижку дня
застосовувати промокоди
автоматично підраховувати загальну вартість покупки
🚀 Основні функції
🛒 Відображення товарів

Список доступних товарів із цінами:

T-shirt
Jeans
Jacket
Sneakers
Hoodie
🎯 Вибір товарів
Користувач обирає товар за номером
Ввід 0 завершить покупку
📅 Знижка дня
Вводимо дату у форматі dd.mm
Програма обирає один товар зі знижкою -10%
🎟 Промокоди
Промокод	Знижка
SALE10	-10%
SALE20	-20%
SUPER30	-30%

Промокод застосовується до всієї суми покупки

💰 Підрахунок суми
Перевірка коректності введення
Додавання ціни товарів до загальної суми
Вивід фінальної вартості
🛠 Технології
Мова: C++
Стандарт: C++17
Бібліотеки: iostream
📂 Структура проєкту
project/
└── clothingstore.cpp
▶️ Як запустити
Скомпілювати програму
Запустити виконуваний файл
Ввести дату (формат dd.mm)
Обирати товари за номером
Ввести 0 для завершення покупки
Ввести промокод (або no)
